# 🛰️ SensorMeteo v1.0

**SensorMeteo** — это продвинутая система уличного мониторинга микроклимата. Проект объединяет в себе сеть беспроводных датчиков на базе Wemos D1 Mini и центральный сервер на Raspberry Pi 5 с интеграцией в Telegram и получением внешних метеоданных.

---

## 🚀 Основные возможности
*   **Локальный мониторинг**: Получение температуры, влажности и давления с высокоточных датчиков BME280/680.
*   **Telegram Bot**: Управление системой через удобные команды (`/mysens`, `/getweather`, `/setregion`).
*   **Умный прокси**: Встроенный туннель **Xray (VLESS Reality gRPC)** для стабильной работы Telegram в условиях сетевых ограничений.
*   **Внешние данные**: Получение точного прогноза погоды через Open-Meteo API по координатам пользователя.
*   **Гибкая маршрутизация**: Использование **FRP** для доступа к серверу датчиков из любой точки мира.

---

## 🏗 Архитектура системы

1.  **Датчики (Nodes)**: Wemos D1 Mini + BME680. Отправляют данные по HTTP POST на центральный сервер.
2.  **Сервер (Backend)**: Python (Flask) + SQLite на Raspberry Pi 5. Принимает, хранит и обрабатывает данные.
3.  **Связь (Network)**: 
    *   **Входящие**: FRP пробрасывает порты 25580 (данные) и 25569 (управление).
    *   **Исходящие**: Xray (SOCKS5 на 25590) обеспечивает боту выход в Telegram API.

## 🛠 Установка и запуск

### 1. Серверная часть (Raspberry Pi)
Убедитесь, что у вас установлен Python 3.10+ и необходимые библиотеки:
```bash
pip install pyTelegramBotAPI requests flask pandas
```

Запуск Xray (используйте ваш конфиг VLESS):
```bash
sudo systemctl start xray
```

Запуск бота:
```bash
python3 main.py
```

### 2. Прошивка датчиков (Wemos D1 Mini)
Используйте Arduino IDE. Требуются библиотеки:
*   `Adafruit_BME680`
*   `Adafruit_BME280`
*   `ESP8266WiFi`
*   `ESP8266HTTPClient`

В коде укажите ваш `sensorId` и `serverUrl` (белый IP вашего VPS или локальный IP малинки).

## 🤖 Команды бота
*   `/start` — Регистрация и первичная настройка координат.
*   `/mysens` — Текущие данные со всех ваших домашних датчиков.
*   `/getweather` — Прогноз погоды по региону (выбор интервала кнопками).
*   `/setregion` — Обновление координат (широта, долгота).
*   `/addsens` — Привязка нового датчика по ID.
*   `/delsens` — Удаление датчика из вашего списка.
